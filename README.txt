This directory contains experimental materials from the following paper:

Wagers, M.W., Phillips, C. (2014). Going the distance: memory and control processes in active dependency construction. Quarterly Journal of Experimental Psychology, 67(7), 1274-1304. http://dx.doi.org/10.1080/17470218.2013.858363

There are three files, each corresponding to one of the three experiments. The materials files are in LINGER format (Rohde, 2003, http://tedlab.mit.edu/~dr/Linger/). Lines beginning with hash-mark name the experiment, the item number and the condition. Lines beginning with a question-mark give a comprehension question (comprehension questions only available for Experiment 3).
